## **Getting Started**

### **Clone the Repository**

```sh
git clone https://github.com/CVamsi27/cookie-clicker
cd cookie-clicker
```

### **Create .env file with values**

- MONGO_URI = Your Mongo DB URL
- VITE_API_URL = Backend URL

### **Start backend**

```sh
cd backend
npm install
npm run dev
```

### **Start frontend**

```sh
npm install
npm run dev
```
